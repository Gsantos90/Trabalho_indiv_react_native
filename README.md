# Crunchyroll Clone

Este projeto é um clone visual do aplicativo Crunchyroll, feito em React Native.

## Funcionalidades
- Banner com destaque de anime
- Blocos de cards de animes e filmes com scroll lateral
- Barra de navegação inferior
- Layout inspirado no app original

## Como rodar
1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o projeto:
   ```bash
   npx expo start
   ```

3. login e senha
   Estão sem autenticação, então basta colocar qualquer um que irá acessar.



## Observações
- Este projeto é apenas para fins de estudo e demonstração visual.
- Não possui integração com a API oficial da Crunchyroll.
